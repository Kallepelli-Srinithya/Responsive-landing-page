# Responsive Landing Page

A professional and visually appealing landing page developed using HTML, CSS, and JavaScript. The project features a fixed interactive navigation bar, smooth scrolling, animated UI elements, glassmorphism design, hover effects, and responsive layouts for all screen sizes.

## Features

- Interactive fixed navigation menu
- Navbar style changes on scroll
- Smooth scrolling navigation
- Modern glassmorphism UI
- Animated hero section
- Hover effects and transitions
- Scroll reveal animations
- Fully responsive design
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
